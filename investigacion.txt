
Análisis comparativo de modelos de Machine Learning y 
Series de Tiempo para la optimización del pronóstico de demanda de inventarios en empresas comercializadoras

Bruno Valentin Vigil Mogollon
Albert Raul Urcuhuaranga Cachay


Taller de Investigación Sistemas

2026



Objetivo general y específicos 

Objetivo general:

Determinar el modelo de predicción con mayor nivel de precisión mediante el análisis comparativo 
entre algoritmos para  optimizar el pronóstico de demanda de inventarios en una empresa comercializadora.

Objetivo específico:

Diagnosticar el estado actual de los datos históricos de ventas de la organización
para identificar patrones y valores anormales.

Implementar modelos de series de tiempo y algoritmos de machine learning
utilizando lenguajes de programación de código abierto.

Evaluar el rendimiento de los modelos implementados mediante el uso de métricas de error estadístico


Resumen

Esta investigación aborda la ineficiencia en la gestión de inventarios del sector comercial,
donde los pronósticos imprecisos generan pérdidas por stock inmovilizado o desabastecimiento. 
El objetivo es determinar el modelo de predicción más preciso mediante un análisis comparativo entre 
algoritmos de Machine Learning y Series de Tiempo.
La metodología comprende el diagnóstico de datos históricos para 
identificar patrones y la implementación de modelos en Python.
El rendimiento se medirá a través de métricas de error estadístico (MAE y RMSE),
permitiendo identificar la tecnología con mayor confiabilidad.
El resultado final proporcionará una base científica para mejorar la toma de decisiones estratégicas.
